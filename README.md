# TexCan Capital Website - SVG Logo Final

This version uses the uploaded logo file and rebuilds the site so the logo is sharp, transparent, and not cut off.

## What changed

- Uses `texcan-logo.svg` for header, hero, and footer.
- No white logo background.
- Logo has padding inside the SVG so it does not clip.
- Keeps the high-resolution skyline and home photos.
- Stats section removed.
- Root-level files for GitHub Pages.

## Files to upload

Upload everything in this folder to the GitHub repo root:

- `CNAME`
- `index.html`
- `styles.css`
- `style.css`
- `script.js`
- `README.md`
- `texcan-logo.svg`
- `texcan-logo-original.svg`
- `texcan-logo-fallback.png`
- `favicon.png`
- `hero-skyline.png`
- `about-home.png`

## After upload

1. Choose **Replace** when GitHub asks.
2. Commit the changes.
3. Wait 1–2 minutes for GitHub Pages to redeploy.
4. Hard refresh:
   - Mac: `Command + Shift + R`
   - Windows: `Ctrl + F5`
