## Upload steps

1. Download and unzip this package.
2. Open your GitHub repository.
3. Click **Add file** → **Upload files**.
4. Drag all files from this package into GitHub.
5. Choose **Replace** when prompted.
6. Commit changes.
7. Wait 1–2 minutes.
8. Hard refresh:
   - Mac: `Command + Shift + R`
   - Windows: `Ctrl + F5`

Do not delete `CNAME`.
